## 1. 사이트 맵

```
becaneparis.com/
├── / (메인 - 컬렉션 쇼케이스)
├── /indexes/products (ALL - 전체 상품)
├── /looks/heroines-drop (Look 상세 페이지 x 11)
├── /story/* (Stories x 4)
│   ├── /story/lou-escobar-becane-paris
│   ├── /story/coco-capitan
│   └── /story/jean-dusartel ...
├── /products/* (개별 상품 상세)
├── /page/terms-of-service
├── /page/shipping-policy
└── /page/size-guide
```

## 7. 인터랙션 상세 분석

### 7.1 장바구니 사이드 패널 (`cart-panel.png`)

**트리거**: 헤더의 `CART 00` 클릭

**UI 변화**:
```
┌─────────────────────────────────────────────────────────────────┐
│  BECANE     ALL 27  STORIES 04              CLOSE 00           │
├─────────────────────────────────────────────────────────────────┤
│  YOUR CART IS EMPTY                                             │
│                                                                 │
│  (빈 상태 - 상품 없음)                                           │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                    [배경: 페이지 전체 흐리게 처리]                │
│                    (opacity 낮춤, 클릭 불가)                     │
└─────────────────────────────────────────────────────────────────┘
```

**관찰된 특징**:
- `CART` → `CLOSE`로 텍스트 변경
- 사이드 패널이 아닌 **인라인 확장** 방식
- 배경 콘텐츠 흐리게(dim) 처리
- 빈 장바구니: "YOUR CART IS EMPTY" 메시지
